# AI 分析

> 免登录访问

## 功能概述

AI 分析页面是 OpenQmt 的核心特色功能，通过对接大语言模型，提供对话式的股票与市场分析能力。

## 主要功能

- **多模型支持**：可选择 OpenAI、DeepSeek、通义千问、Anthropic、Google Gemini、Ollama 等服务商
- **流式响应**：采用 SSE 协议消费流式响应，实时展示 AI 输出
- **对话管理**：支持创建多个对话、切换、删除，历史记录持久化
- **页面助手**：顶部工具栏的 AI 助手按钮，可快速发起分析

## 模型配置

在「系统设置 → 模型配置」中：

- 各提供商独立配置 API Key 和 API 地址
- 支持启用 / 禁用单个模型
- 支持添加自定义模型和自定义提供商
- 系统提示词可自定义（默认为金融分析助手）

## 使用方式

1. 在左侧菜单中点击「AI 分析」（或点击「新建对话」）
2. 在顶部下拉菜单选择模型服务商和具体模型
3. 在输入框中输入你的问题
4. AI 将以流式方式实时返回分析结果

::: tip 免登录
AI 分析功能无需登录即可使用。只需在系统设置中配置好 API Key 即可开始对话。
:::

## 技术实现

- 前端通过 Tauri HTTP 插件发起请求
- 支持 OpenAI 兼容的 API 接口
- SSE（Server-Sent Events）实现流式响应消费
- 对话历史通过 tauri-plugin-store 持久化
