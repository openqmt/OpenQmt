# 股市行情

> 免登录访问

## 功能概述

股市行情是 OpenQmt 的核心功能之一，涵盖 A 股大盘指数、个股详情和板块行情。

## 大盘详情

点击大盘指数卡片进入详情页，可查看：

- **分时图**：实时分时走势与成交量柱状图
- **逻辑复盘**：A股复盘、美股复盘、热点要闻、龙虎榜单等 Tab
- **板块数据**：各板块涨跌排行

## 个股详情

输入股票代码或名称搜索，进入个股详情页可查看：

- 实时价格、涨跌幅、成交量
- 分时走势图（含成交量柱状图）
- 评分与情绪指标（温度 / 估值 / 情绪）
- 逻辑复盘 Tab 配置

## 板块详情

从大盘详情页点击板块名称，进入板块详情页查看板块内个股列表与排行。

## 价格卡片规范

PriceCard 组件统一展示行情数据：

- 最高价 / 最低价根据涨跌动态着色
- 情绪指标（温度 / 估值 / 情绪）以标签形式展示
- 暗色主题下红色保持正红色 `#ff0000`

## 使用方式

1. 在左侧菜单中点击「股市行情」
2. 在搜索框中输入股票代码或名称
3. 点击搜索结果进入个股详情
4. 或在大盘详情页浏览板块与个股
