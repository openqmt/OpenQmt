# 系统设置

> 需要登录访问

## 功能概述

系统设置区域包含四个子模块，提供 OpenQmt 的深度配置能力。

## 模型配置

配置大语言模型提供商，支持 AI 分析功能。

### 预置提供商

| 提供商        | 说明               |
| ------------- | ------------------ |
| OpenAI        | GPT 系列模型       |
| Anthropic     | Claude 系列模型    |
| Google Gemini | Google AI 模型     |
| DeepSeek      | 深度求索模型       |
| 通义千问      | 阿里云 AI 模型     |
| Ollama        | 本地部署的开源模型 |

### 配置项

- **API 地址**：各提供商独立的接口地址
- **API Key**：各提供商独立的密钥
- **模型管理**：启用 / 禁用单个模型，添加自定义模型
- **自定义提供商**：可添加完全自定义的模型服务商
- **系统提示词**：自定义 AI 助手的系统角色设定

## 推送通知

配置市场行情推送通知，支持多渠道推送。

- 通知类型管理
- 推送渠道配置（飞书、企业微信、钉钉、WxPusher 等）
- channels 字段为 `string[]` 类型

## 功能控制

管理主页菜单的可见性和排序。

- **可见性管理**：勾选 / 取消勾选控制菜单项显示
- **排序管理**：通过上下箭头调整菜单顺序
- 隐藏的菜单项仍可通过设置区域访问

## 个人中心

查看和管理个人账户信息。

- 用户信息展示
- 账号管理
