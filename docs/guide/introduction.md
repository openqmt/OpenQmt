# 项目介绍

## 什么是 OpenQmt

OpenQmt 是一个**基于 AI 的股票分析投资工具**。

它的核心理念是：**简单**。没有复杂的逻辑，不追求大而全，只做最实用的投资辅助功能。

## 技术栈

| 技术       | 说明                           |
| ---------- | ------------------------------ |
| Tauri 2    | 跨平台桌面应用框架，轻量高性能 |
| Vue 3.5+   | 渐进式前端框架                 |
| TypeScript | 类型安全的开发体验             |
| Naive UI   | 现代化 Vue 3 组件库            |
| Pinia      | 直观的 Vue 状态管理            |
| ECharts    | 数据可视化图表                 |
| Vite 6     | 极速构建工具                   |

## 核心功能

- **黄金行情** — 实时黄金品种价格
- **股市行情** — A 股大盘、个股、板块行情
- **基金排行** — 基金收益排行与详情
- **认知学习** — 金融投资知识库（免登录访问）
- **AI 分析** — 大模型对话分析股票与市场（免登录访问）
- **系统设置** — 多模型配置、推送通知、功能控制

## 访问策略

OpenQmt 采用宽松的访问策略，大部分核心功能无需登录即可使用：

| 功能     | 是否需要登录 |
| -------- | ------------ |
| 黄金行情 | 否           |
| 股市行情 | 否           |
| 基金排行 | 否           |
| 认知学习 | 否           |
| AI 分析  | 否           |
| 个人中心 | 是           |
| 系统设置 | 是           |
| 推送通知 | 是           |
| 功能控制 | 是           |

## 设计哲学

> 简单，不意味着简陋。而是在保持核心价值的同时，去掉一切不必要的复杂性。

- 代码结构清晰，无冗余
- 界面简洁，操作直观
- 功能聚焦，不堆砌
