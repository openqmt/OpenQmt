# 功能总览

OpenQmt 提供 6 大核心功能模块，覆盖投资分析的各个方面。

## 功能一览

| 功能                   | 说明                         | 是否需要登录 |
| ---------------------- | ---------------------------- | ------------ |
| [黄金行情](./gold)     | 实时黄金品种价格行情         | 否           |
| [股市行情](./stock)    | A 股大盘、个股、板块详情     | 否           |
| [基金排行](./fund)     | 基金收益排行与详情           | 否           |
| [认知学习](./learn)    | 金融投资知识库               | 否           |
| [AI 分析](./ai)        | 大模型对话式分析             | 否           |
| [系统设置](./settings) | 模型配置、推送通知、功能控制 | 是           |

## 界面布局

OpenQmt 采用经典的侧边栏 + 内容区布局：

```
┌──────────────────────────────────────┐
│  侧边栏  │       顶部工具栏          │
│          ├──────────────────────────┤
│  Logo    │                          │
│  菜单    │       内容区域            │
│          │                          │
└──────────┴──────────────────────────┘
```

- **侧边栏**：Logo 区点击可返回主页行情，菜单项支持折叠、排序与隐藏
- **顶部工具栏**：显示当前页面标题、搜索框、AI 助手、主题切换、登录/用户信息
- **内容区域**：各功能页面的主要内容

## 主题切换

支持亮色 / 暗色双主题，一键切换。暗色主题下红色保持正红色，保证涨跌色彩一致性。

## 移动端适配

全面适配移动端，侧边栏自动转为抽屉式，搜索框自适应折叠，确保在手机上的操作体验。
