# 基金排行

> 免登录访问

## 功能概述

基金排行页面提供基金收益排行与详情查看，帮助用户筛选和了解基金产品。

## 主要功能

- **收益排行**：按收益率排序展示基金列表
- **基金搜索**：在搜索框中输入基金代码或名称快速查找
- **基金详情**：点击基金进入详情页，查看净值走势、持仓、业绩等

## 数据来源

基金数据来自天天基金等公开接口，通过环境变量配置数据源地址。

## 使用方式

1. 在左侧菜单中点击「基金排行」
2. 在搜索框中输入基金代码或名称
3. 点击基金进入详情页查看完整信息
