# 社区交流

## 社区规则

为了营造良好的交流氛围，请遵守以下规则：

1. **尊重他人** — 保持友善，禁止人身攻击、歧视或骚扰性言论
2. **内容相关** — 交流内容应围绕投资、股票分析、AI 应用等相关主题
3. **禁止广告** — 未经允许不得发布商业广告或垃圾链接
4. **真实客观** — 分享真实经验和观点，禁止传播虚假信息或恶意误导
5. **遵守法规** — 遵守国家法律法规，不讨论敏感政治话题
6. **保护隐私** — 不得泄露他人或自己的个人隐私信息
7. **禁止作弊** — 禁止使用自动化工具刷帖、灌水等行为

> 违反社区规则的发言将被删除，严重者将被永久封禁。
